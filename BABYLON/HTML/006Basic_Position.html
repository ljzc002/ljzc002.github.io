简介

在这个教程中，我们将学习怎样放置、旋转、缩放在以前的教程里建立的所有种类的物体。我们也会建立一个父物体来开始理解父物体如何影响（子物体）的位置、旋转和缩放。

Playground示例场景 3 - 位置，旋转和缩放立方体

怎样做 ?

像以前一样，我们在createScene方法开始时将一些基本图形元素引入场景中 :

function createScene() {
    var scene = new BABYLON.Scene(engine);
    var light = new BABYLON.PointLight("Omni", new BABYLON.Vector3(0, 100, 100), scene);
    var camera = new BABYLON.ArcRotateCamera("Camera", 0, 0.8, 100, new BABYLON.Vector3.Zero(), scene);
    }

然后建立你需要数量的立方体:

//建立三个立方体
    var box1 = BABYLON.Mesh.CreateBox("Box1", 6.0, scene);
    var box2 = BABYLON.Mesh.CreateBox("Box2", 6.0, scene);
    var box3 = BABYLON.Mesh.CreateBox("Box3", 6.0, scene);
    […]
  • 位置

你可以方便的把立方体放置在场景里，任何你喜欢的位置。举个例子:

//放置立方体
    box1.position = new BABYLON.Vector3(-20,0,0);
    box2.position.x = -10; // or box2.position = new BABYLON.Vector3(-10,0,0);
    box3.position.x = 0;

现在，通过同样简单的编码，你可以旋转立方体任意角度。举个例子：

  • 旋转 (角度用弧度表示)
//绕x轴旋转立方体
    box1.rotation.x = Math.PI/4; // or box1.rotation = new BABYLON.Vector3(Math.PI/4,0,0);

    //绕y轴旋转立方体
    box2.rotation.y = Math.PI/6;
  • 缩放

是的，沿着任意坐标轴缩放立方体同样很简单。举个例子：

//沿着x轴拉伸至2倍（是物体自身坐标系的X轴）
    box3.scaling.x = 2; // or box3.scaling = new BABYLON.Vector3(2,1,1);
  • 父物体

另一种Babylon特性在影响着一个物体，与其他物体的相对性，通过在两个网格之间建立父子连接。这种连接暗示了父物体的变化（位置、旋转、缩放）也会被应用到子物体上。

//使立方体3相对于立方体1放置（立方体3的父物体是立方体1，可以看到立方体3的定位方式由相对于世界坐标变成了相对于立方体1的自身坐标，立方体1的拉伸和旋转效果也同样作用在了立方体3上）
    box3.parent = box1;
    box3.position.z = -10;

注意

父子层级被所有框架所考虑。所以在指定子物体之前的所有父物体变化也会被应用到子物体上。

所有建立父子连接之后的变化，将会被子物体相对于父物体保持。

为求简单，不要旋转或移动一个子物体直到你把它和父物体关联好，同时要记住在父物体的旋转应用到子物体上之后，再添加到子物体的旋转将相对于父物体。
